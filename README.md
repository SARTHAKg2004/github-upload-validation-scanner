# 🔍 GitHub Upload Validation Scanner

A Python-based static code analysis tool that scans GitHub repositories to detect file upload validation vulnerabilities in frontend and backend code.

## 🚀 Features
- Detects file upload inputs
- Identifies missing validation (security risks)
- Extracts allowed file types (pdf, jpg, png)
- Classifies severity (High, Medium, Low)
- Generates Excel report

## 🛠 Tech Stack
- Python
- GitPython
- Pandas
- Regex

## ▶️ How to Run
```bash
python scanner.py
